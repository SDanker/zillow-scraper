# Zillow Web Scraper

This project is a web scraping script written in Python that scrapes Zillow.com for house listings in Los Angeles. The script extracts the address and price of each house and stores the data in a CSV file.

## Installation

To install the required dependencies, run the following command:

```bash
pip install -r requirements.txt
```

## Usage

1. Clone this repository to your local machine.
2. Install the required dependencies as mentioned above.
3. Run the web scraping script:

```bash
python zillow_scraper.py
```

The script will create a CSV file named `zillow_listings.csv` containing the address and price of houses in Los Angeles.
